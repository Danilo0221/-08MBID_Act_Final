Trabajo Practico - 08MBID Visualización de datos

Introducción

A partir de la elaboración de este trabajo se dará explicación a la problematica de noticias criminales por delito registrados en el Sistema Penal Oral Acusatorio en la Ley 906 de 2004 y Ley 1098 de 2006 desde hechos ocurridos en 2010, esto se hará por medio del uso de dos herramientas de visualización de datos, como lo son Power BI y Python utilizando la librería seaborn.

Fuente de datos

Utilizaremos un set de datos en formato csv tomado del portal Datos Abiertos, el cual es un lugar web en Colombia que permite el acceso a miles de conjuntos de datos de caracter público y de libre uso de diferentes tematicas. De allí tomamos un set de datos que muestra el total de noticias criminales por delito registrados en el Sistema Penal Oral Acusatorio en la Ley 906 de 2004 y Ley 1098 de 2006 desde hechos ocurridos en 2010.


Url de los datos: https://www.datos.gov.co/Justicia-y-Derecho/Conteo-de-Procesos/q6re-36rh

Objetivos

  1. Crear los suficientes objetos visuales para dar un entendimiento de la problematica de los crimenes registrados en colombia y la gestión en cada uno de ellos.

  2. Contrastar por medio de las diferentes herramientas de visualización la interacción de las gráficas y lo que representan.

  3. Fomentar el uso de herramientas de interacción cloud, por medio de reportes de uso publico como de este tipo.
